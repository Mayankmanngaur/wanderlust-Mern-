# Wanderlust

A full-stack web application inspired by Airbnb where users can browse, create, edit, and manage travel listings.

## Live Demo

🔗 [View Live Project](https://wanderlust-mern-six.vercel.app)

## Features

* User Authentication & Authorization
* Create, Edit and Delete Listings
* Image Uploads with Cloudinary
* Reviews and Ratings System
* Interactive Maps Integration
* Flash Messages for User Feedback
* Responsive UI

## Tech Stack

### Frontend

* EJS
* Bootstrap
* HTML
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

### Other Tools

* Passport.js
* Cloudinary
* Mapbox
* Joi Validation

## Installation

```bash
git clone <repository-url>
cd wanderlust
npm install
```

Create a `.env` file and add the required environment variables.

Run the project:

```bash
node app.js
```
